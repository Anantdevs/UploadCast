# Learncast

Learncast is a simple React-based application for uploading, visualizing, and interacting with audio files. It offers an intuitive and responsive interface to enhance the audio experience, with features like waveform visualization, a resizable timeline, and more.

## Features

- **Audio Upload:** 
  - Users can upload audio files using Multer, supporting multiple audio formats.
- **Drag-and-Drop Upload:** 
  - Integrated Dropzone feature for easy drag-and-drop file upload.
- **Animated UI:** 
  - Smooth and responsive animations using Tailwind CSS and Locomotive Scroll.
- **Waveform Visualization:** 
  - Real-time audio waveform display, with a bar to adjust the waveform size.
- **Resizable Timeline:** 
  - Allows users to zoom in and out of the audio timeline for better control.

## Technologies Used

- **React:** 
  - The core framework for building the application.
- **Tailwind CSS:** 
  - Utility-first CSS framework for styling.
- **shadcnUI:** 
  - For pre-built and customizable UI components.
- **Multer:** 
  - Middleware for handling file uploads on the server.
- **Locomotive Scroll:** 
  - Provides smooth scroll animations and parallax effects.
- **Waveform Library:** 
  - (Specify the exact library if used) For rendering the audio waveform.

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/anantdevs/UploadCast.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd UploadCast
   cd client
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Run the development server:**
   ```bash
   npm start
   ```

The application should now be running on `http://localhost:3000`.

## Usage

1. **Upload an Audio File:**
   - Drag and drop an audio file into the designated area or click to upload.
2. **Visualize the Audio:**
   - View the waveform and use the timeline to navigate through the audio.
3. **Adjust the Waveform:**
   - Use the bar below the waveform to increase or decrease its size for better precision.
